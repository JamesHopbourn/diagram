目录操作：
ls      显示文件 (简略显示：ls、详细显示：ls -la)
pwd     显示路径 (pwd)
mkdir   新建目录 (mkdir 目录名)
cd      切换路径 (cd ..、cd -、cd ~、cd /path/to/dir)
clear   历史清屏 (clear)
tree    树状显示 (tree -N /path/to/dir)

文件处理：
touch   新建文件 (touch test.txt)
cat     查看文件 (cat test.txt)
echo    标准输出 (echo 'hello'、echo 'line 1' > test.txt、echo '' > test.txt、echo 'line 2' >> test.txt)
grep    匹配字符 (grep -n 关键词 文件名、grep -rn 关键词 目录名)
nl      显示行号 (nl 文件名)
cp      文件复制 (cp test test_backup、cp -r * /path/to/dir)
mv      文件移动|重新命名 (mv 原文件名 新文件名 | mv 文件名 /path/to/dir)
zip     解压缩包 (打包文件：zip 压缩包名.zip 文件名、打包目录：zip -r 压缩包名.zip 目录、解压：unzip 压缩包名.zip)
vim     代码编辑 (编辑模式：i、从下行开始编辑：o、退出编辑模式：esc、保存：w、退出：q、移动到文件首行：gg、移动到文件尾行：G、移动到行首：^、移动到行尾：$、下一个括号：%、移动到指定行：行号、打开指定行：vim 文件名 +行号、文本搜索：/text、整行删除：dd、撤销：u、U)
git     代码管理 (仓库克隆：git clone、增加修改文件：git add -A、查看文件状态：git status、添加提交备注：git commit -m '提交备注'、推送远程仓库：git push、撤销本地修改：git reset --hard)
python  运行脚本 (python 文件名.py)
gcc     代码编译 (gcc test.c -o test)
open    打开文件 (open 文件名、open 目录名)
md5     哈希计算 (md5 文件名)
history 历史命令 (history)
rm      删除文件|删除目录 (rm 文件名 | rm -rf 目录名)
chmod   更改权限 (赋予文件可执行权限：chmod +x 文件名)

符号指令：
*       所有文件 (ls *)
*.txt   所有 txt 文件 (以此类推 png、py、sh......)
$()     调用变量 (echo $(SHELL))
&       后台运行 (brew install vim &)
>       输出重定向 (echo 'hello' > new.txt、echo '' > new.txt)
>>      输出追加内容 (echo 'hello' >> new.txt)
;       顺序执行 (命令前后没有逻辑关系直接执行 vim test.py ; python test.py)
|       管道符号 (前面命令执行结果输入后面命令 history | grep cat)
&&      条件执行 (前面命令执行成功执行后面命令 vim test.py && python test.py)
||      条件执行 (前面命令执行失败执行后面命令 python test.py || vim test.py) 

快捷键：
control+A 行首
control+E 行尾
control+D 退出
control+C 停止
